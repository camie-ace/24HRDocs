## Overview

The `AddTicketModal` component provides a comprehensive multi-step form for creating roadside assistance tickets. It supports both insurance and fleet management organizations, with features for policy validation, service selection, service provider assignment, and more.

## Installation

```bash
npm install react-hook-form react-phone-number-input country-state-city axios
```

## Props Interface

```typescript
interface AddTicketProps {
  showTrigger?: boolean;        // Show the "Add Ticket" button trigger
  openState?: boolean;           // External control of dialog open state
```
